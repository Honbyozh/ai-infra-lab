# AI Infra 自学实验室

这是一个为期 32 周的 AI Infra 自学项目，覆盖模型服务、AI Platform、vLLM、Kubernetes、GPU 调度、性能优化、分布式推理、NCCL、CUDA 与 Triton。

## 项目周期

- 启动日期：2026-07-27（周一）
- 计划完成：2027-03-07（周日）
- 总周期：32 周 / 224 天
- 计划投入：384 小时，平均每周 12 小时
- 每周节奏：周一至周五每天 1 小时，周六 5 小时，周日 2 小时

## 开始学习

1. 按 [启动前准备计划](docs/startup-preparation-plan.md) 完成 2026-07-23 至 2026-07-26 的启动门禁。
2. 启动前按 [学习前准备指南](docs/pre-study-preparation.md) 配置“本机学习 + 实验机执行”的双机工作流并完成知识热身。
3. 在 [32 周逐日学习计划](AI-Infra-32-Week-Daily-Learning-Plan.md) 中找到当天任务和验收条件。
4. 开始前从 [每日学习记录模板](docs/templates/daily-log.md) 新建当天记录，写明计划时长与目标。
5. 在规定时长内学习和实验，将代码、命令输出、截图或测试结果保存到仓库。
6. 按 [执行与验收规则](docs/execution-guide.md) 完成自测；四项统一条件全部通过后才可标记完成。
7. 更新 [项目进度表](docs/progress.md)，并在当天记录中写明实际用时、证据、未决问题和下一步。

建议的日志文件名：

```text
docs/logs/2026-07-27-w01-d1.md
```

首周日期和每日任务入口见 [Week 1 启动清单](docs/week-01-kickoff.md)。硬件或环境暂不具备时，不直接视为完成；请登记到 [延后补验台账](docs/deferred-validations.md)。

## 完成定义

每日任务必须同时满足：

- 能复现：从新终端按记录可得到相同结果。
- 有证据：保留代码、测试输出、指标、截图或笔记。
- 能解释：可以不看资料回答当天关键问题。
- 有边界：写出至少一个限制、失败场景或未决问题。

只完成阅读或达到计划时长，不等于通过验收。
